Descripción de las pruebas automatizadas

Este proyecto utiliza Selenium para realizar pruebas automatizadas en un sitio web de compras en línea. A continuación, se describe cada prueba realizada y el resultado exitoso obtenido:

Test1: Navegar a la página web
Esta prueba consiste en abrir el navegador web y dirigirse a la página principal del sitio web de compras en línea.

Resultado: La prueba se completó exitosamente. Se abrió el navegador y se cargó la página principal del sitio web sin ningún problema.

Test2: Buscar Productos mediante un nombre
En esta prueba, se ingresa un término de búsqueda en el campo de búsqueda del sitio web para buscar productos relacionados.

Resultado: La prueba se completó exitosamente. Se ingresó el término de búsqueda "Monitor" en el campo de búsqueda y se enviaron los resultados correctamente.

Test3: Seleccionar producto deseado
Una vez que se muestran los resultados de la búsqueda, se selecciona un producto específico de la lista de resultados para ver más detalles.

Resultado: La prueba se completó exitosamente. Se seleccionó un producto de la lista de resultados de búsqueda y se accedió a la página de detalles del producto sin problemas.

Test4: Agregar producto deseado al carrito
Después de ver los detalles del producto, se agrega el producto deseado al carrito de compras.

Resultado: La prueba se completó exitosamente. Se hizo clic en el botón "Agregar al carrito" y el producto fue agregado al carrito sin errores.

Test5: Entrar al carrito donde se encuentra nuestro producto deseado
Finalmente, se hace clic en el enlace del carrito de compras para acceder al carrito y verificar que el producto agregado esté presente.

Resultado: La prueba se completó exitosamente. Se hizo clic en el enlace del carrito y se accedió al carrito de compras. El producto agregado previamente se mostró correctamente en el carrito.
